- ### 概要
  2022年12月中旬から2023年1月末にかけてゲームエンジンUnityを用いて,徳島大学ゲームクリエイトプロジェクトのメンバ数名で共同制作した「市松模様」というゲーム作品です.

- ### ゲームコンセプト
  敵を倒して衣装を集めて,自分だけの着せ替え人形を作ろう！

- ### プレイ人数
   1人

- ### プレイ時間
   1回の探索で約2分

- ### 使用機器
   PC(キーボード＆マウス操作)

- ### ゲームルール
   #### 1.探索
       - 髪の毛を伸ばして,敵を攻撃
       - 倒した敵の着ていた衣装がドロップ
       - ドロップした衣装を入手
   #### 2.キャラメイク
       - これまでに入手したアイテムを市松人形に着せよう
   #### 3.図鑑
       - これまでに集めた衣装を図鑑で確認できる
  
- ### 操作説明
   Wキー：前進
  
   Sキー：後退
  
   Aキー：左移動
  
   Dキー：右移動
  
   マウス：視点操作
  
   スペースキー：髪の毛を伸ばす
